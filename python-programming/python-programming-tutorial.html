python programming language tutorial

Python language is high level and interpreted programming language. It is also support Object Oriented programming(OOP), which approach to develop applications. It is simple and easy to learn. Python provides a lots of high-level data structures.

History of Python programming language
1.Python language was developed by Guido van Rossum at the National Research Institute for Mathematics and Computer Science in Netherlands.

2.Python language is derived from many other programming languages including ABC, Modula-3, C, C++, Algol-68, SmallTalk, and Unix shell etc.

3.It is now maintained by a core development team of institute and Guido van Rossum still holds a role in directing its progress.

Features of Python language
1.Interpreted:-It is processed at run time by the interpreter. programmer does not need to compile his program before executing it. This language is similar to PERL and PHP.

2.Interactive Mode:- This language supports an interactive mode, which allows interactive testing and debugging of the snippets of the code.

3.Object-Oriented:- It supports Object-Oriented programming that encapsulates code within objects.

4.Standard library:- Python has a bulk of the library is very portable and cross-platform compatible on Windows, UNIX etc.

5.Portable:- It can run on a large number of the hardware platforms and has the same interface on all platforms.

6.Extendable:- Programmer can add low level modules to python interpreter. These types of modules enable programmers to add or customize their tools to be more efficient.

7.Databases:- Python language provides interfaces to a large amount of commercial databases.

Application of python language
1.Web development (server-side)
2.Software development
3.Mathematics
4.System scripting.
5.Audio or Video based Applications
6.3D CAD Applications
7.Console Based Application
