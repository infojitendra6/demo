introduction to java programming tutorial

Introduction to java programming :-It is a high-level programming language, which is developed by Sun Microsystems lab and It is released in 1995. It runs on a large number of platforms, like Windows, Mac OS, and various versions of UNIX.

History of java programming language
James Gosling, Mike Sheridan and Patrick Naughton started Java language project on June 1991. The team of java language project is called Green Team.
At starting time, it is designed for small electronic appliances like set-top boxes, televisions etc.
At starting time, its file extension was .gt. and it was called "Greentalk"

why use java programming language ?
1.Object Oriented:-It is Object Oriented programming language, which can be easily extended. java language is based on the Object model.

2.Platform Independent:- It does not depends on platform, Because Its compiler converts the source code to bytecode, which is Intermidiate Language. The bytecode can be executed on any type of platform using JVM.

3.Simple:- It is easy to learn. If students understand the basic concept Object Oriented programming, then it would be master of java.

4.Multithreaded:- Java language has multithreaded feature, which can perform a large number of tasks at a same time. The feature of java allows to construct interactive applications that can be run easily.

5.Interpreted:- Its byte code is translated on the fly to native machine instructions, which is not stored anywhere. In this way the development process is more rapid and analytical.

6.High Performance:- Java language use Just-In-Time(JIT) compilers, in this way Java enables high performance.

7.Dynamic:- Java language is considered to be more dynamic in nature than C or C++ language. Java programs can carry extensive amount of run-time information, which can be used to verify and resolve accesses to the objects on run-time.
