for loop in c programming language

In for loop in c programming, Executes a statement or group of statements while a given condition is true. It tests the condition before executing the program
The for loop syntax in C is given below:-

for(initialization;condition;incr/decr)
{  
//code to be executed  
when condition is satisfied.
}

First Step :- First step initialization happens and the counter variable gets initialized.
Second Step :- In the Next step the condition is checked. Where the counter variable is tested for the given condition. If the condition returns true in program then the C statements inside the body of for loop gets executed. If the condition returns false in program then the for loop gets terminated and the control comes out of the loop.
third Step: - After successful execution of statements inside the body of loop. the counter variable is incremented or decremented depending on the operation (++ or â€“) used in program.

There are following three types of loops used in the C language.
Types of Loops in C
1. while loop
2. do while loop
3. for loop

1.while loop in C language
Repeats a statement or group of statements while a given condition is true. It tests the condition before executing the program.
The syntax of while loop is given below :-

while(condition){  
//code to be executed
 when condition is satisfied.  
}

2.do-while loop in C program
It is more like a while statement, except that it tests the condition at the end of the loop body.
The syntax of do-while loop is given below :- 

do{  
//code to be executed 
when condition is satisfied. 
}
while(condition);

3. for loop in C language
Executes a sequence of statements multiple times and abbreviates the code that manages the loop variables.
The syntax of for loop is given below:-

for(initialization;condition;incr/decr)
{  
//code to be executed  
when condition is satisfied.
}

Loops Control Statements in C

Loop control statements change execution from its normal sequence. When execution leaves a scope, all automatic objects that were created in that scope are destroyed.

There are following control statements in C.
1 break statement :-
Terminates the loop or switch statement and transfers execution to the statement immediately following the loop or switch.
2 continue statement
Causes the loop to skip the remainder of its body and immediately retest its condition prior to reiterating.
3 goto statement
Transfers control to the labeled statement.

Advantage of loops in C
1) It provides code reusability.
2) Using loops, we do not need to write the same code again and again in program.
2) Using loops, we can traverse over the elements of data structures (array or linked lists).
