while loop in c programming

While loop in c is also known as a pre tested loop, which repeats a statement or group of statements, until given condition is true in program. While loop tests the condition before executing the Code of program.
Flowchart of while loop in C programming

The syntax of while loop is given below :-

while(condition){  
//code to be executed
 when condition is satisfied.  
}

Now print table of 5 using while loop example in C

#include <stdio.h>
int main ()
{
   /* Here i is defined as local variable */
   int i = 5;
   // while loop execution 
   while( i <=50 ) {
      printf("print value of i : %d\n", i);
      i=i+5;
   }
   return 0;
}

Output after compilation and execution of program is given below.

print value of i : 5
print value of i : 10
print value of i : 15
print value of i : 20
print value of i : 25
print value of i : 30
print value of i : 35
print value of i : 40
print value of i : 45
print value of i : 50

Explanation of above program
Step 1 :- When i is 5, test condition i <= 50 is true. Hence, the body of the while loop is executed and print 5 on the screen, then the value of i is increased to 10.

Step 2 :- Now i is 10, the test condition i <= 50 is again true. The body of the while loop is executed again and print 10 on the screen, then the value of i is increased to 15.

Step 3 :- This process goes on until i becomes 55. When i is 55, the test condition i <= 50 will be false, then the loop terminates.
