goto statement in c programming

Goto statement in c language is used to transfer the program control to a given label. It is also know as jump statement in c language. It can be used to repeat some part of program for a particular condition.

Flowchart of goto statement in C programming

The syntax of a goto statement in C:-

goto label;
......
......
......
......
label: statement;

Label can be any plain text except C keyword and it can be set anywhere above or below to goto statement in the program.

1. program of goto statement:-

#include <stdio.h>
int main ()
{
int i, j, k;    
for(i=0;i<10;i++)  
{  
  for(j=0;j<5;j++)  
  {  
   for(k=0;k<3;k++)  
   {  
   printf("print value of i, j, k : %d %d %d\n",i,j,k);  
   if(j == 3)  
    {  // jump control on label
    goto label;   
    }  
  }  
 }  
}  
  label:   
  printf("now control out from loop");   
}

Output after compilation and execution of program is given below.

print value of i, j, k : 0 0 0
print value of i, j, k : 0 0 1
print value of i, j, k : 0 0 2
print value of i, j, k : 0 1 0
print value of i, j, k : 0 1 1
print value of i, j, k : 0 1 2
print value of i, j, k : 0 2 0
print value of i, j, k : 0 2 1
print value of i, j, k : 0 2 2
print value of i, j, k : 0 3 0
now control out from loop
