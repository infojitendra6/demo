continue statement in c programming

The continue statement in C is used to bring the control of program to the beginning of the loop. It skips some lines of code inside the loop and continues with the next iteration in program.
Flowchart of continue statement in C programming

The syntax of a continue statement :-

continue;

Now write continue statement program using for loop in C

The for loop syntax in C is given below:-

for(initialization;condition;incr/decr)
{  
//code to be executed  when condition is satisfied. 
if(condition)
{
continue;
}
//code
}

program of for loop using continue statement :-

#include <stdio.h>
int main ()
{
 int i=1;   
for(i=1;i<=10;i++){      
if(i==5){
/* if value of i is equal to 5,
then skip code and it will continue the loop */    
continue;    
}    
printf("print value of i : %d \n",i);    
} 
return 0; 
}

print value of i : 2 
print value of i : 3 
print value of i : 4 
print value of i : 6 
print value of i : 7 
print value of i : 8 
print value of i : 9 
print value of i : 10
