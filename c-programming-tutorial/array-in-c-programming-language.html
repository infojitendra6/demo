Array in c programming language

Array in c programming:- It is defined as the collection of similar type of data like int, float, char etc, which stored at contiguous memory locations. It is derived data type in C language which can store the primitive type of data such as int, char, double, float etc. Array has also the capability to store the collection of derived data types such as pointers, structure etc.

Arrays consist of contiguous memory locations in ROM. Its the lowest address corresponds to the first element of array and the highest address to the last element of Array.

The syntax of array in c programming is given below :-

data_type array_name[array_size];

Declaration and Initialization of array are given below :-

int marks[10];   //Declaration of array
int marks[5]={20,30,40,50,60}; //Declaration with Initialization

Array programs in c

#include <stdio.h>
int main() {
 int i;
 int marks[10]={50, 86, 58, 68, 58, 69, 42};
 for (i=0; i<=6; i++) {
 printf("Address arr[%d] is %p and its value is %d\n", i, &marks[i], marks[i]); 
}
 return 0;
}

Properties of Array in C:-
1.Each element of array is same type of data and same size i.e. int = 4 bytes, char=1 bytes.
2.The elements of array are stored at contiguous memory locations in ROM. Starting address of array is called the base address.
3. Array name represents the base address of array. It is the address of the first element of array.
4.The index of array starts with 0 and ends with N-1. where, N stands for the number of elements in array. For Example, there is an integer array of 6 elements, then itâ€™s indexing will be 0 to 5.

Advantages and Disadvantages of array in c
Advantages:-
1.It is collection of similar types of data.
2.If we want to store the marks of students it will be easy to store in array otherwise we have to store in different different location in memory, which is not easy to memorise .
3.It is also used to implement other data structure like linked lists, stack, queue , graph etc.
Disdvantages:-
1.The time complexity of array is increase in insertion and deletion operation, so it is not useful in insertion and deletion operation.
2.The wastage of the memory in the array because it is fixed in size.
3.It is not possible to increase the size of array in compilation time
