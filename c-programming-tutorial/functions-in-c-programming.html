functions in c programming language

Functions in c programming is a group of statements which perform a task together . Every C program has at least one function, which is called main function, and all the most trivial programs can define additional functions in program.

Advantages of functions in C
By using functions, we can avoid rewriting the same code again and again in program.
We can call a functions any number of times in program and from any place in program.
We can track a large program easily using function.

Types of Functions in c programming
1.Library Functions
2.User-defined functions

1.Library Functions in c programming
Library Functions are the functions which declared in the C header files such as scanf(), printf(), gets(), puts(), ceil(), floor() etc. are library function.

To use library function in program we include corresponding header file using the preprocessor directive #include. For example to use input and output functions like printf() , scanf() we include studio.h, for math library function we include math.h , for string library function string.h should be included.

Library functions examples

#include <stdio.h>
#include <math.h>
int main ()
{    
float num, root;
   printf("Enter a number: ");
   scanf("%f", &num);
   // Computes the square root of number and stores its value root.
   root = sqrt(num);
   printf("\nSquare root of %.2f = %.2f", num, root);
   return 0;
}


Output of program after compilation and execution is given below.

Enter a number: 16 
Square root of 16.00 = 4.00
Enter a number: 20
Square root of 20.00 = 4.47
Enter a number: 81 
Square root of 81.00 = 9.00

User-defined functions in c
User-defined functions are the functions which created by programmer, he/she can use it many of times in program. It reduces the complexity of a big program and make easy of program..
To create and use these type of functions, before it we should know about these three things given below:-
1. Function definition
2. Function declaration
3. Function call
Before discussion these three point let us write a simple program.

User-defined functions examples

#include 
int sum(int x, int y);  //Function declaration 
int main ()
{    
int a, b, s;
   printf("Enter any two number: ");
   scanf("%d%d", &a,&b);
   s=sum(a,b);  //Function call 
   printf("\nsum of %d and %d is %d", a,b,s);
   return 0;
}
int sum(int x, int y)
{     //Function definition 
    int s;
    s=x+y;
    return s;
}

Output of program after compilation and execution is given below.

Enter a number: 56 67 
sum of 56 and 67 is 123
Enter a number: 2456 677
sum of 2456 and 677 is 3133

1. Function definition
Function definition consists of the whole description of the function. It tell what the function is doing and what is its input and output in program. Function definition consists of two part - a function header and a function body.

The syntax of Function definition:-

return_type function_name( parameter list ) {
   body of the function
}

2. Function declaration
Function declaration tells the compiler about a function name and how to call it in program.

The syntax of Function declaration:-

return_type function_name( parameter list );

3. Function call
When the program calls a function, the program control is transferred to the called function. A called function performs a defined task in the program and when its return statement is executed or when its function-ending closing brace is reached, then it returns the program control back to the main program.

The syntax of Function call:-

function_name( argument list);
