Do while loop in c programming

Do while loop in C is just like a while loop, except that it tests the condition at the end of the loop body. It is also called a post tested loop in c programming language.

do while loop syntax is given below :- 

do{  
//code to be executed 
when condition is satisfied. 
}
while(condition);

We know that conditional expression appears at the end of the loop in the statements, so the statement(s) in the loop executes once in the program before the condition is tested.

In the program if the condition is true, the flow of control jumps back up to do and the statement(s) in the loop executes again. This process repeats until the given condition becomes false in given code.
Flowchart of do while loop in C programming

Now print table of 6 using do while loop example in C

#include <stdio.h>
int main ()
{
   /* we define local i */  
   int i = 6;
    /*start do while loop execution */ 
   do {
      printf("print value of i: %d\n", i);
      i = i + 6;
   }while( i <=60 );
   return i;
}
