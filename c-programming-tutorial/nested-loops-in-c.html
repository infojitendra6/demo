Nested loops in c programming

When one loop is inside another loop, this type of loops is called nested loops in c programming language.
There are following nested loop in C given below
Flowchart of while loop in C programming

1.nested for loop :-when one for loop inside another for loop, then it is called nested for loop
The syntax of a nested for loop in C is given below :-

for ( init; condition; incr/decr ) {
   for ( init; condition; incr/decr ) {
      statement(s);
   }
   statement(s);
}

Now print star pattern using nested loop example in C

#include <stdio.h>
int main ()
{
int i, j;
 for(i=1; i<=6; i++)
  {
     /* Print i number of stars */
   for(j=1; j<=i; j++)
   {
     printf("*");
   }
   /* Move to next line */
   printf("\n");
  }
 return 0;
}

Output of program after compiletion and execution of above program is given below.

*
**
***
****
*****
******

nested while loop :-when one while loop inside another while loop, then it is called nested while loop
The syntax of a nested while loop in C is given below :-

while(condition) {

   while(condition) {
      statement(s);
   }
   statement(s);
}

Now print star pattern using nested while loop example in C

2.nested do while loop :-when one do while loop inside another do while loop, then it is called nested do while loop
The syntax of a nested do while loop in C is given below :-

do {
   statement(s);
   do {
      statement(s);
   }
   while( condition );
}
while( condition );
