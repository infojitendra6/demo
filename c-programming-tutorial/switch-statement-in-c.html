switch statement in C programming

TheSwitch statement in C is an alternate method of if else ladder statement, which allows to execute multiple operations for many different possibles values in program for a single variable called switch statement.

Rules for switch statement in C language
1.Its expression must be an integer or character type.
2.The case value of switch statement must be an integer or character.
3.The switch statement can have an optional default case in program, which must appear at the end of the switch statement.

Flowchart of switch statement in C programming

The syntax of switch statement:-

switch(expression) {
case value 1  :
statement(s);
break;  /* optional */ 
case value 2  :
statement(s);
break;  /* optional */ 
.......
.......
.......
.......
default :  /* Optional */ 
statement(s);
}

switch statement without break keyword in c programming

The break statement in switch case is optional. If there is no break statement found, then all cases will be executed after the matched case.

The syntax of if else ladder statement in C:-

switch(expression) {
case value 1:
statement(s);
case value 2:
statement(s);
.......
