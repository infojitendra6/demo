Nested if statement in c programming

The nested if statement in C is inside of another if statement in program, It means an if statement that target of another if statement.

Flowchart of nested if statement in C programming

The syntax of nested if statement in C:-

if( condition 1) {
  code executes, when condition 1 is true */ 
  if(condition 2) {
  /* code executes, when condition 2 is true */ 
  else {
  /* code executes, when condition 2 is false */ 
 }
 }

  /* code executes, when condition 1 is false */ 
 }
}

if else ladder statement in c programming language

The if else ladder statement in C is used to check the set of conditions in sequence in program. An if condition is tested if and only if, when all previous if conditions in if-else ladder is false.

Flowchart if else ladder statement in C programming

The syntax of if else ladder statement in C:-

if (condition 1) 
{
   // statement(s) 
}

    // statement(s)
