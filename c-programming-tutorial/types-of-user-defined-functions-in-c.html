Types of user-defined functions in c

The user-defined functions are classify into four different type on the basis of the return value and the arguments.
There are following type of user-defined functions in c

1.Function with no argument and no return value
Function with no argument means the called function, which does not receive any data from calling function and Function with no return value means called function, which does not return any data to the calling function.

Functions that have no argument and no return value are written as:-

void func(void);
int main(void)
{
.......
func();
......
return 0;
}
void func(void)
{
......
}

Function no argument and no return value example

#include <stdio.h>
void area(void);   //function prototype
int main()
{
  area();    //function call
  return 0;
}
void area(void)
{
  float side, area;
  printf("Enter the side of square :");
  scanf("%f",&side);
  area= side*side;
  printf("\nArea of Square = %f",area);  
}

Output of program after compiletion and execution is given below.

when input is 10 
Enter the side of square : 10
Area of Square = 100.000000
when input is 20.5
Enter the side of square : 20.5
Area of Square = 420.250000
