Local variable and global variable

Local variable:- The variable that defined within the body of a function or a block are called local variable to that function or block. These type of variables are created when function is entered and these type of variables are destroyed when the function is exited.

The syntax of Local variable in c is given below :-

void func(void);
{ 
 int a, b;
 ......
 ......
}

Here a and b are local variables of function, which are defined within the body of the function func(). This type of variable can be used only in those functions or blocks, in which they are declared.

Local variable example

#include <stdio.h>
void func(void);
int main () {
  int a=10, b=10, c;  /* local variables in main function  */ 
  c = a + b;
  printf ("The value of a = %d, b = %d and c = %d in main function\n", a, b, c);
 func();
  return 0;
}
void func(void)
{ 
 int a=20, b=30, c;  /* local variables in User-defined functions  */ 
  c = a + b;
printf ("The value of a = %d, b = %d and c = %d in User-defined functions \n", a, b, c);
}

Output of program after compilation and execution is given below.

The value of a = 10, b = 10 and c = 20 main function
The value of a = 20, b = 30 and c = 50 in User-defined functions

Global variable:- The variables that are defined outside of function are called global variables. All functions in program can used and modify these type of variables. It is useful to declare a global variable, if it is to be used by many functions in program. Global variables are automatically initialized to 0 at declaration time of variable.

Global variable example

#include <stdio.h>
void func1(void);
void func2(void);
int a, b=6;
int main() {
  printf("Inside main() : a=%d, b=%d\n", a, b);
  func1();
  func2();
  return 0;
}
void func1(void)
{
  printf("Inside func1() : a=%d, b=%d\n", a, b);   
}
void func2(void)
{
  int a=8;
  printf("Inside func2() : a=%d, b=%d\n", a, b);
}

Output of program after compilation and execution is given below.

Inside main() : a=0, b=6
Inside func1() : a=0, b=6
Inside func2() : a=8, b=6

There are following Difference between local variable and global variable in C

Local variable Global variable
It is declared inside of function. It is declared outside of function.
If it is not initialized, then it store a garbage value. If it is not initialized, then it store zero value as default value.
This type of variable is created when the function starts execution and lost when the functions terminate in program. Global variable is created before main function of program and lost when the program terminates.
Parameters passing is required for local variables to used the value in other functions. Parameters passing is not necessary for a global variables.
When the value of local variable is modified in one function, then changes are not be affected in another function. When the value of global variable is modified in one function, then changes are affected in another function.

Advantages and disadvantages of Local variables.
Advantages:-
You can give same name of local variables in different functions, because they are only recognized by function in which they are declared.
Local variables are deleted as soon as function is terminated and release the memory space which occupies by it.
Disdvantages:-
In local variable common data required to pass repeatedly, because data sharing is not possible between functions.
They have a very limited scope in program.

Advantages and disadvantages of Global variables.
Advantages:-
1.You can use the global variable in all functions or modules of program.
2.Global variable is useful when multiple functions are using same data.
Disdvantages:-
1.When a lot variables declared as global variable, then they remain in the memory till program execution is completed. This can be cause of out of Memory issue.
2.In this type of variable data can be modified by any function in program. In this way any statements written in the program can change the value of global variables.
