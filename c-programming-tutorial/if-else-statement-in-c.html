if else statement in c programming

The if else statementin C is used to perform operations based on a given condition in the program. The operations specified in program, if block are executed, when given condition is true.

There are the following type of if statement in C language.
1. If statement
2. If-else statement
3. If else-if ladder
4. Nested if

If statement in c programming language

The if statement is work according to the Boolean expression. If the Boolean expression is true , then the block of code is executed. If the Boolean expression is false, then the block of code is not executed .

Flowchart of if statement in C programming

The syntax of if statement in C:-

if(boolean_expression) {
   /* code will execute if the boolean
 expression is true, otherwise skip */
}

1. program of if statement in C:-

#include <stdio.h>
int main ()
{
int i = 30;
   if( i < 40 ) {
      /* if condition is true then print the following code */
      printf("i is less than 40\n" );
   }
   printf("value of i is : %d\n", i);
   return 0;
}

Output after compilation and execution of program is given below.

i is less than 40
value of i is : 30

If else statement in c programming language

The if else statementis used to perform two different operations in a single given condition. If the Boolean expression is true, then if block will be executed. If the Boolean expression is false, then else block will be executed.

Flowchart of if else statement in C programming

The syntax of if statement in C:-

if(boolean_expression) {
   /* code will execute if the boolean expression is true */  
} else {
     /* code will execute if the boolean expression is false */  
}

1. program of if else statement in C:-

#include <stdio.h>
int main ()
{
int i = 50;
   if( i < 40 ) {
        /* if condition is true then print the following code */  
      printf("i is less than 40\n" );
   }
    else {
        /* if condition is false then print the following code */  
      printf("i is greater than 40\n" );
   }
   printf("value of i is : %d\n", i);
   return 0;
}

Output of code after compiletion and execution of above program is given below.

i is greater than 40
value of i is : 50
