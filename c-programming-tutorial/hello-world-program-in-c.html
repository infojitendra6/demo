Hello World Program in c programming

We start to study first program of C language. In this we learn how to print “hello world” program, before printing ” hello world”. We study the basic building blocks of C language.

There are following the basic building blocks.

1. Preprocessor Commands
#include is a preprocessor command, which tells C language compiler to include stdio.h file before going to actual compilation.

2. Functions
Every C program has a function, which is called main function. The program execution begins from main function.

3. Variables
A variable is a name of the memory location, which used to store data. Its value can be changed and reused many times.
It is a way to represent memory location through symbol, which can be easily identified.
Example :- int a, float b, char c.

4. Statements & Expressions
Expressions :–It is a instruction of program to be executed and returns a value.
Statements :– It is used to form the sequence of a program in programming language (e.g. if statement , while-do statements). A statement can be simple or complex.

5. Comments
In programming language Comments are used to provide information about the lines of code, which are ignored by the compiler . There are 2 types of comments in the C programming language

Single Line Comments
This type of comments are represented by double slash \\, which is used to single line comments.

2. Multiple Line Comments
Multiple line comments are denoted by /*.........*/.
