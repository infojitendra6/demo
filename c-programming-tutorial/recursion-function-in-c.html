Recursion function in c language

Recursion is a process in which the Function is defined itself. Problem is solved by repeatedly breaking it into smaller problems, which is similar in nature to original problem. The smaller problems are solved and their solutions are applied to get the final solution of original problem.

The syntax of recursion function in c is given below :-

void rec();
int main(void) 
{ 
  .....
  rec();
  .....
} //end of main()
void rec()
{
  .....
  rec();
  .....
} //end of rec()

Factorial using recursion

#include <stdio.h>
long int fact(int n);
int main() {
 int n, ans;
 printf("Enter a number : ");
 scanf("%d", &n);
 if(n<0)
 {
 printf("\nNo factorial for negative number");
 }
 else
 {
 ans=fact(n);
 printf("\nFactorial of %d is %d\n", n, ans);
 }
 return 0;
}
long int fact(int n)
{
 if(n==0)
 return 1;
 else
 return(n*fact(n-1));
}

Output of program after compilation and execution is given below.

when input is 5 
Enter a number : 5
Factorial of 5 is 120
when input is 10
Enter a number : 10
Factorial of 10 is 3628800

Fibonacci series using recursion

#include <stdio.h>
int fib(int n);
int main() {
 int n, i;
 printf("Enter number of term : ");
 scanf("%d", &n);
  printf("\n");
 for(i=0; i< n; i++)
    printf("%d ", fib(i));
 return 0;
}
int fib(int n)
{
 if(n==0 || n==1)
 return 1;
 else
 return(fib(n-1)+fib(n-2));
}

Advantages and disadvantages of Recursion function in C.
Advantages:-
1.Recursion function reduce unnecessary calling of functions in program.
2.Through recursion method, we can Solve problems in easy way while its iterative solution is very complex and big.
3.It can lead to more readable and efficient algorithm descriptions.
4.It is a very useful way to defining objects that have a repeated similar structural form.
Disdvantages:-
1.It takes more processor time in execution of program. It is main disadvantages of recursion function.
2.Recursion uses lot of stack space.
3.This type of function, we must use an if statement somewhere to force the function to return without recursive call being executed, otherwise the function will never return.
