loops in c programming language

Loops in c programming language is defined as repeating the same process multiple times in program, until a specific condition satisfies.

There are following three types of loops used in C language.
Types of Loops in C language
1. while loop
2. do while loop
3. for loop

1.while loop in C language
While loop is defined as Repeating a statement or group of statements, until a given condition is satisfies . It tests the condition before executing the conditional code.
The syntax of while loop in C is given below :-

while(condition){  
//code to be executed
 when condition is satisfied.  
}

2.do-while loop in C program
It is similar to a while loop, except that it tests the condition at the end of the loop body in program.
The syntax of do-while loop is given below :- 

do{  
//code to be executed 
when condition is satisfied. 
}
while(condition);

3. for loop in C language
For loop is defined as executing a sequence of statements multiple times and abbreviates the code that manages the loop variables of program.
The syntax of for loop is given below:-

for(initialization;condition;incr/decr)
{  
//code to be executed  
when condition is satisfied.
}

Loops Control Statements in C

Loop control statements change execution of loops from its normal condition.

There are following control statements in C.
1 Break statement :-
Break statement terminates the loop or switch statement and transfers execution to the statement immediately following the loop or switch statement.
2 Continue statement
Continue statement use to skip the remainder of its body and immediately retest its condition prior to reiterating.
3 Goto statement
Goto statement use to transfers the control of program at defined statement label.

Advantage of loops in C
1.Using loop, we do not need to write the same code many number of time in program.
Using loops, we can traverse over the elements of data structures likes array, linked lists.
