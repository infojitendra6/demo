Cpp programming language tutorial

Cpp programming language :-It is case-sensitive programming language, which supports object-oriented and generic programming. C++ is advance version of C programming language.

History of java programming language
1.Cpp language was developed by Bjarne Stroustrup starting in 1979 at Bell Labs in New Jersey. It is extension of C language and originally named C with Classes but later it was renamed C++ in 1983.

2.It is a middle-level language. C++ is a combination of both high-level and low-level language.
3.C++ programming language was 3 major update in 2011, 2014, and 2017 to C++11, C++14, and C++17.

Object-Oriented Programming
C++ language supports object-oriented programming, There are following four major pillar of the object oriented programming which used in C++.
1.Inheritance
2.Polymorphism
3.Encapsulation
4.Abstraction

Standard Libraries of C++:-
Standard Libraries of C++ is divided into three parts, which are given below:-
1.The core library, which includes data types, variables literals etc.
2.The standard library , which includes the set of functions manipulating strings, files etc.
3.The Standard Template Library, which includes the set of methods manipulating a data structure.
